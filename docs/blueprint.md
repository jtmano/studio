# **App Name**: Fitness Focus

## Core Features:

- Date Selection: Allow users to select the 'Week' and 'Day' to log workouts.
- Workout Logging: Display exercises in distinct blocks, showing weight and reps entered.
- Progress Tracking: A historical view of workout data displayed in a chart or table to visualize progression.
- Routine Saving: The ability to save workout routines for future usage.
- AI Exercise Suggestion: Generate an exercise suggestion based on workout history and desired muscle group. Acts as a tool to provide tailored fitness ideas, enhancing user workout variety and effectiveness.
- Log Workout Data: A 'Log Workout' button that stores entered workout information in a Supabase 'Workout History' table.
- Load Workout Template: Upon entering a Day (1-5), load the corresponding workout template from the Supabase 'Templates' table.

## Style Guidelines:

- Primary color: Black (#000000).
- Secondary color: White (#FFFFFF) for clean backgrounds.
- Accent: Red (#FF0000) for buttons and active elements to draw attention.
- Clean, sans-serif fonts for easy readability on iOS.
- Simple, card-based layout for exercises and data presentation.
- Use minimalist icons to represent different exercises and features.
